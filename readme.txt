■概要
名前の通り、コンピュータにおみくじをやってもらうCGIスクリプトです。
「大吉」「中吉」「吉」「小吉」「凶」「大凶」という画像がランダムに
表示されます。　HPの飾りとしてお使い下さい。
# この程度ならJavaScriptでも出来そうだけど、いちいちこの程度で待た
# されるの嫌いだから作りました(ネスケ使い)

し下さい。　ただし、結果については責任を持てません(笑)


■導入
☆解凍☆
omikuji.lzhを解凍します。

omikuji.cgi   (おみくじ本体)
0〜5.gif      (結果画像)
readme.txt    (これ)


☆設定☆
omikuji.cgiをテキストエディタで開き、一行目のperlへのパスをあなたの入
会しているプロバイダに合わせて変更して下さい。

次に、このおみくじを表示させるHTMLのURLを設定して下さい。
このおみくじはココで設定されたページ以外からは動かない仕組みになってい
ます。　(いたずら防止の為)

ハイ、これで設定終わりネ。　簡単アルね！　私広島生まれは…(以下略)


☆アップロード☆
さて、今度はomikuji.cgiと0〜5.gifをサーバにアップロードします。
これらは全て同じディレクトリに置いて下さいね。
そして、omikuji.cgiのパーミッションを705(-rwx---r-x)に変更します。

(ここでは、 〜/cgi-bin/omikuji にアップロードしたものと仮定します)


☆使い方☆
おみくじを表示させたい位置に
<IMG SRC="./cgi-bin/omikuji/omikuji.cgi" WIDTH=140 HEIGHT=60 ALT="おみくじ">
みたいな感じのタグを埋め込みます。
つまり、扱いは画像と同じです。　横幅や縦幅を変更して使っても良いですし、
これにさらにAタグを被せて他の所にリンクを張ったりも出来ます。

応用としては、0〜5.gifという名前であなたのイラストを保存して、同じディ
レクトリに置いておけばトップ絵をランダムに表示させるということも出来ま
す。　絵を絶対6枚用意しなきゃいけないという欠点はありますが。
